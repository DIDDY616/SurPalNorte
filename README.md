# Pal' Sur del Norte — Rediseño Web

Rediseño moderno del sitio web de Pal' Sur del Norte, organización sin fines de lucro enfocada en comunidades rurales vulnerables del sur de Nuevo León.

## Estructura

```
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos
├── js/
│   └── main.js         # Scripts (nav, scroll reveal, carousels)
└── README.md
```

## Tecnologías

- HTML5 semántico
- CSS3 (variables, grid, flexbox, clip-path, transitions)
- JavaScript vanilla (IntersectionObserver, carousel)
- Google Fonts (Cormorant Garamond + Manrope)

## Despliegue

### Vercel
```bash
npm i -g vercel
vercel
```

### GitHub Pages
Sube los archivos al repo y activa Pages desde Settings > Pages > Source: main branch.

### Netlify
Arrastra la carpeta del proyecto a [netlify.com/drop](https://app.netlify.com/drop).
